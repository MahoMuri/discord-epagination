OtherButtonsOptions

Options for the other buttons (delete, backMain).

Properties:
Name Type Description
enabled boolean

Whether the button should be enabled or not.
position number

Position of the button in the row. You need to set a positive index, the default buttons (back, foward) are in the array and you can't change their position and the back to first page button is the first button that will receive the position and the delete button after. So make sure to set a correct index if you want to do something very specific.
